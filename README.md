# PI-Squared-


Have you gotten this Error when you want to deposit SET into your Fastset wallet from Arbitrium Network?

Here are the steps on how to solve this problem.

On your transaction menu via Metamask, click on the edit icon beside the " network fee"

Select Advance 

Edit the Max Base fee and Priority Fee to " 0.1 "
Click on SAVE 

Then click confirm to approve the transaction.

You will get this message "Deposit To Source Chain Is Completed" 

Enjoy navigating Pisquared.
